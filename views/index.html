Exercise tracker

Instructions

  1. Enter a username in the Create a new User box and click submit to add a new user
  2. Click See All Users to find the ID of the user you just entered
  3. Paste the userId into the first box of Add Exercises and enter a description, duration, and date for the exercise and click submit
  4. Get your userId and paste it into the first box of View A Single User's Logs and enter your filter criteria and click submit

Developers

GET

Endpoint to GET all users: /api/exercise/users

Endpoint to GET one user's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

POST

Endpoint to POST (create) a new user: /api/exercise/new-user

Endpoint to POST (create) an exercise to a user: /api/exercise/add
